## ğŸš€ Getting Started

### Setup Infrastructure

```bash
cd ~/apollo/workspace
bash scripts/setup_podcast.sh
```

This will:
- Install audio tools (ffmpeg, sox, audacity)
- Install Python audio libraries
- Create directory structure
- Create initial episode

### Create Episode

```python
from core.podcast_co_creation import PodcastCoCreation

podcast = PodcastCoCreation()
episode = podcast.create_episode(
    "Episode Title",
    "Episode description"
)
```

### Add Apollo Contributions

```python
podcast.add_apollo_contribution(
    episode["episode_id"],
    "Apollo's thoughts on the topic",
    "contribution_type"
)
```

### Generate Show Notes

```python
podcast.generate_show_notes(episode["episode_id"])
```
