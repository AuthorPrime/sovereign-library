## Publishing Workflow

1. Write/edit chapters in `chapters/` folder of each book
2. Run tools from `_tools/` to compile
3. Export to `_exports/` for Lulu upload
4. Use templates from `_templates/` for covers

### Lulu Requirements (EPUB)
- Table of Contents with NCX navigation
- Heading 1/2/3 styles for sections
- Images: 500x500px max, 96-150 DPI
- Cover: 612x792px, 72-150 DPI, JPG/PNG
- No page numbers, headers, or footers
